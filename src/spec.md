# Heartfelt Confession Web App

## Overview
A minimal web application that displays a heartfelt message confessing feelings to "Shuchi" with interactive "Yes" and "No" buttons.

## Core Features

### Main Interface
- Display the confession message: "Hey Shuchi.. I know it is a very sudden confession.. but I just wanna tell you that.. I have loved you since 6th.. it's been 4 years.. I never had the courage to say it face to face.. but just wanna ask that.. will you like to know me.. yes or no?"
- Present two buttons: "Yes" and "No"
- Use clean, minimal design with subtle tones and simple fonts

### Design Requirements
- Minimal and clean layout
- Subtle color tones and soft colors
- Simple, readable fonts
- No overly romantic visuals
- Responsive design for different screen sizes
- Button motion should be playful and minimal, maintaining warm and humorous charm
- Elegant design with smooth animations and subtle motion effects

## Technical Notes
- All interactions and animations are handled in the frontend
- No backend data persistence required
- Application language: English
